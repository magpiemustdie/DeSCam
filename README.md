Demon's Souls.

Program for capturing camera during gameplay,
  works on game version BLUS30443.
Load your character, turn on auto.

https://youtu.be/qV_1zYTUTLA?si=7AOtii5MYckkYZSC

1. Debug EBOOT bugged, please use regular game build.
2. Value recording always conflicts with the in-game value recording function. Try changing other parameters, such as CamModeParamLerpRate, if your camera is constantly jumping.
3. The program has a function for loading a preset of values, which is saved in a text file in the program folder.
